This page contains links to other sites with information either directly about PCB or about tools which may be of interest to users of PCB.

  • harry eaton's PCB Homepage.
  • The PCB Sourceforge Project Summary Page. Thats where you can find information on how to submit bug reports and how to access the most recent sources via CVS (in the `pcb' module).
  • A PCB Package (stable release) for NetBSD.
  • A PCB Package (development version) for NetBSD.
  • Bill Wilson's gsch2pcb tool for interfacing gEDA and PCB.
  • GNU EDA (gEDA). The gEDA suite can, among other things, provide a schematic capture front end for PCB.
  • Stephen Meier's Land Pattern Creation for Thomas Nau's and Harry Eaton's PCB.
